Mod pLua

The Apache server has Lua integrated into its architecture this is covered on the Mod Lua page. Mod Lua is primarily intended for server configuration its secondary function allows you to write dynamic web pages in pure Lua. A similarly named third party module pLua, which allows both pure and mixed HTML/Lua code extends Lua’s capability.

The Uniform Server Zero provides a seperate plugin: ZeroXI_lua_1_0_0 to support pLua. The following are included in this plugin:

  • Adds additional pages to this default manual. (Configuration details, examples)
  • Adds Lua Apache configuration file C:\UniServerZ\core\apache2\conf\extra\us_plua.conf
  • Adds New folder C:\UniServerZ\core\us_lua
  • Adds module lfs.dll

The plugin covers basic web-applications using pLua as the scripting language.
